# ExpenseTracker-MERN stack

Full Stack App built using MERN(Mongodb Express React Node)
CRUD operations covered

To initailze the repo locally 
```
npm install
cd client
npm install
cd ..
```
To run server on local machine
```
 # Run front and backend
 npm run dev
 
 # Backend only
 npm run server
 
 # Frontend only
 npm run client
 ```
